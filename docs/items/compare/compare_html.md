# Html 常用字符转义

| 字符         | 十进制    | 转义字符 |
| ------------ | --------- | -------- |
| <            | `&#60;`   | `&lt;`   |
| >            | `&#62;`   | `&gt;`   |
| "            | `&#34;`   | `&quot;` |
| &            | `&#38;`   | `&amp;`  |
| 不断行的空格 | `&#160;`  | `&nbsp;` |
| 全角的空格   | `&#8195;` | `&emsp;` |
| 半角的空格   | `&#8194;` | `&ensp;` |
