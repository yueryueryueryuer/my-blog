# 常用对照表

## Html 常用字符转义

| 字符         | 十进制   | 转义字符 |
| ------------ | -------- | -------- |
| <            | \&#60;   | \&lt;    |
| >            | \&#62;   | \&gt;    |
| "            | \&#34;   | \&quot;  |
| &            | \&#38;   | \&amp;   |
| 不断行的空格 | \&#160;  | \&nbsp;  |
| 全角的空格   | \&#8195; | \&emsp;  |
| 半角的空格   | \&#8194; | \&ensp;  |

## RGB 颜色

| 中文名称 | 英文名称 | 色号    | R.G.B       |
| -------- | -------- | ------- | ----------- |
| 白色     | White    | #FFFFFF | 255 255 255 |
| 黑色     | Black    | #000000 | 0 0 0       |
| 红色     | Red      | #FF0000 | 255 0 0     |
| 绿色     | Green    | #00FF00 | 0 255 0     |
| 蓝色     | Blue     | #0000FF | 0 0 255     |
